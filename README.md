#### 特点

- 像选电影票一样进行选座
- 快速预约，为考研人保驾护航
- 个人数据展示

#### 系统介绍

系统分为管理员（web）和用户端（微信小程序）。

文件大致分三个部分：

- server （后端）
- miniProject（小程序端）
- vue（前端）
